# Electron React
#### 结合Electron 和 React 开发Windows和MacOS应用
##### 技术栈: Javascript, Electron, React, Ant Design, Scss, Typescript, React Router Dom, NodeJs.

#
#### 安装
``` javascript
npm install
```

#
#### 在浏览器中开发页面，支持热更新
``` javascript
npm run start
```

#
#### 编译
``` javascript
npm run build
```

#
#### 直接启动app
``` javascript
npm run el
```

#
#### [推荐]先编译完成，再启动app
``` javascript
npm run app
```
